weather forecast

This project was about typography and composition. We had to choose a weather report and organize it in several differnt way. It was about creating negative space and understanding the depth that weight and size from trypography can weigh on a piece of work. It was also about learning how to pair off different typfaces, which ones worked together and which ones did not. Aside from the oraganized works we were required to make a creative one. The idea was to use type as image. The creative one was fun to make because the weather report I chose was during a volcanic eruption so I dot to get a bit playful with the letters. I strived to make the letters look like a cloud of smoke.
